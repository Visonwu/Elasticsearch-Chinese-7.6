# meta

附加到字段的元数据。 此元数据对Elasticsearch不透明，仅对在相同索引上工作的多个应用程序共享有关字段（例如单位）的元信息有用



```json
PUT my_index
{
  "mappings": {
    "properties": {
      "latency": {
        "type": "long",
        "meta": {
          "unit": "ms"
        }
      }
    }
  }
}
```

> 注：
>
> 字段元数据最多强制执行5个条目，键的长度小于或等于20，值是长度小于或等于50的字符串。
>
> 字段元数据可通过提交映射更新来更新。 更新的元数据将覆盖现有字段的元数据。
